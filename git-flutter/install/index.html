Install

The git flutter extension can be installed on macOS, Linux or Windows.

macOS and Linux

To install or update git-flutter on your MacBook or Linux machine:

  1. Run the installation script:
 curl -o- https://innersource.flutter.com/git-flutter/install.sh | bash
  1. Close and reopen your terminal to start using git flutter or run the following command to use it immediately:
 [ -d "$HOME/.git-flutter/bin" ] && PATH="$HOME/.git-flutter/bin:$PATH"
  1. Verify the extension installation via git:
 git flutter

Windows

To install on a Windows machine:

  1. Download the latest release v1.8.3.

Or if you have curl installed, use this command:

curl https://innersource.flutter.com/git-flutter/git-flutter_v1.8.3_Windows_x86_64.zip
  1. (Optional) Validate the binary by comparing the provided checksum with that of the download which you can calculate at the command prompt:
CertUtil -hashfile git-flutter_v1.8.3_Windows_x86_64.zip SHA256

  1. Extract the git-flutter_v1.8.3_Windows_x86_64.zip file and copy the git-flutter.exe into C:\Program Files\Git\usr\bin.

  2. Verify the extension installation via git:

git flutter
