GitHub Action Workflows

Use GitHub Action Workflows to validate any proposed code change. This provides fast, automated and transparent feedback to the contributor.

SemVer Git Tags

Use SemVer git tags to trigger the CI pipeline. This is a concise, familar to communicate release and breaking change risk to your users.

Registry Publish

Publish the build to a registry close to the required deployment(s). This may be in AWS, Artifactory or in GitHub Packages as required.
